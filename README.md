# 🍽 Food Recommendation System

## 🚀 Overview
This project is a content-based food recommendation system that suggests similar food items based on cuisine and ingredients.

## 🧠 Features
- Content-based filtering
- Cosine similarity
- Flask API deployment
- Real-time recommendations

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Flask

## ▶️ Run the Project
pip install -r requirements.txt
python app.py

## 🔗 API
http://127.0.0.1:5000/recommend?food=Pizza

## 📌 Example Output
["Pasta", "Burger", "Fried Rice"]
